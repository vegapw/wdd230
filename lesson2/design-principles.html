Contrast

Netflix

netflix.com

On this page, we can see that the background is full of images of movies, but there is a small blur between the images and the text in white, highlighting the white text over the background, that's where we can find the contrast design principle.

Aligment

Udemy

udemy.com

On this page, we can see how everything is aligned and there is a balance between the images and text, that's how we can see the alignment design principle.

White space and clean design

LinkedIn

linkedin.com

On this page, we can see how each item on page has its personal space, there is a significant white space between them, which makes each item more visible and readable without overwhelming the view, that's how we can see the design principle applied.
